# rails-cli

## Overview
Learning project for rails-cli.

## Features
- Command line interface for Rails applications

## Technology Stack
- Ruby on Rails

## Usage
1. Clone the repository
   ```bash
   git clone https://github.com/username/rails-cli.git
   cd rails-cli
   ```
2. Install dependencies
   ```bash
   bundle install
   ```
3. Run the application
   ```bash
   rails server
   ```
